Sección 1.1: Hoja de estilo externa

Se puede aplicar una hoja de estilo CSS externa a cualquier número de documentos HTML colocando un elemento en cada documento HTML. El atributo rel de la etiqueta link debe establecerse en "hoja de estilo" y el atributo href en la ruta relativa o absoluta a la hoja de estilo. Si bien el uso de rutas de URL relativas generalmente se considera una buena práctica, también se pueden usar rutas absolutas. En HTML5, el atributo de tipo se puede omitir . Se recomienda que la etiqueta link se coloque en la etiqueta head del archivo HTML para que los estilos se carguen antes que los elementos a los que diseñan. De lo contrario, los usuarios verán un destello de contenido sin estilo.
