Section 1.2: Internal Styles

CSS enclosed in style tags within an HTML document works like a style sheet external, except that it lives in the HTML document that you design rather than in a separate file, and therefore it can only be applied to the document you are living in. Please note that this item must be inside of the head element for HTML validation (although it will work in all current browsers if placed on the body).



Section 1.4: Inline Styles

Use inline styles to style a specific element. Note that this is not optimal. I know recommends placing style rules in a style tag or in an external CSS file to maintain a distinction between content and presentation. Inline styles override any CSS in an external style sheet or style tag. While This can be useful in some circumstances, this fact often reduces the ability to maintenance of a project.
